Marrow Master

This project introduces a novel technique to automate bone marrow cell classification by combining powerful machine learning algorithms with a Flutter-based mobile app. The fundamental goal is to achieve high accuracy in cell type identification, simplifying hematological diagnosis, increasing speed and accuracy, and reducing pathologists' burden.

  • Robust AI Model: Trained on a large dataset of labeled microscopic cell images, achieving a loss of 0.2286, accuracy of 0.9418, validation loss of 0.4207, and validation accuracy of 0.8856.

  • Mobile Application: Developed with Flutter, this app enables healthcare providers to upload bone marrow cell images for real-time classification.

  • Tools & Techniques: Google Colab, Python, CNN, MobileNet, Classification, Kaggle, Flutter, Keras, TensorFlow, Firebase, Flask API, GitHub.

  • Project Achievements: Finalist in the Google Solution Challenge 2024, 1st place in the Build with AI Idea Competition at GDSC CUI ATD, Runner-Up in the SparkPitch Idea Competition at Tech Vision, Islamia College.

Vehicle Maintenance Prediction System

Implemented a machine learning-based vehicle maintenance prediction system that uses real-time data from an OBDII reader to assess whether a vehicle requires maintenance. This project involved the development of multiple classification models and a mobile app to make predictions accessible and actionable for users.

  • Implemented various classification models (Logistic Regression, SVC, Random Forest, and LGBM) to predict vehicle maintenance needs based on real-time data from an OBDII reader.

  • Optimized model performance using specific hyperparameters, including penalty and C for Logistic Regression, kernel and C for SVC, and various parameters for Random Forest and LGBM.

  • Achieved high accuracy with the LGBM model (87.6%) and evaluated performance using metrics such as accuracy, precision, recall, F1-score, confusion matrix, mean squared error, mean absolute error, and R score.

  • Integrated real-time vehicle data from the OBDII reader into a FlaskAPI server, which processes the data and makes predictions on vehicle maintenance needs.

  • Developed a mobile app that allows users to view real-time predictions and vehicle data, providing an accessible interface for users to monitor their vehicle's condition.
